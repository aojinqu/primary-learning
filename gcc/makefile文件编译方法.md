# MakeFile文件编译方法

听我说谢谢你

https://blog.csdn.net/weixin_42653531/article/details/100159377

简要介绍一下

1. gcc编译器下载，[TDM-GCC](http://tdm-gcc.tdragon.net/download)

2. 然后装环境变量

3. 打开cmd，切换到以上的目录，输入

 ```
  mingw32-make
 ```

会得到可执行文件.exe

执行即可
